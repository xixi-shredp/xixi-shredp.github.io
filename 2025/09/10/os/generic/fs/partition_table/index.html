分区表 Partition Table

目前常用的分区表有 MBR 分区表和 GPT 分区表

MBR 分区表 (Master Boot Record)

  • 存储在硬盘的第一个扇区上(分区表的大小是固定的)
  • 兼容性比较好
  • 最多只能识别2TB的空间
  • 最多只能支持4个主分区或3个主分区+1个扩展分区(逻辑分区不限制)

GPT 分区表 (GUID Partition Table)

  • 支持2TB以上的大硬盘
  • 在Windows系统下可以支持128个主分区
  • 在GPT分区表头中可自定义分区数量的最大值(GPT分区表的大小不是固定的)
